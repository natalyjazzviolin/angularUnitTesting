    Setup event flow#

    1. The parent component is listening for an event on the child component.

    2. When that event is raised by the child, the parent component will call the appropriate method.

    Get a handle on all child components#

    info

    A component is a subclass of a directive.

    By.directive will find elements that have components on them, as well as directive decorators and attributes.

    // Within it()const childComponents = fixture.debugElement    .queryAll(    By.directive(ChildComponent)    )
// Check that the event-trigegred method is calledspyOn(fixture.componentInstance, 'mockMethod');
// Now we can call the eventchildComponents[0].query(By.css('button')).triggerEventHandler('click', { mockMethod: () => {} })
// Expect the method to be called with specific dataexpect(fixture.componentInstance.method)    .toHaveBeenCalledWith('expected data');
